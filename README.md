# Piece It Together

**Piece It Together** is a jQuery-based sliding puzzle web game where players rearrange shuffled image tiles to restore the original picture. The game includes multiple difficulty levels, support for custom image uploads, and automatic game state saving using `localStorage`.

This project was created during the 4th semester of university studies.

## Features

- Sliding puzzle mechanics with 3x3, 4x4, and 5x5 grids
- Choose from built-in images or upload your own
- Game state is automatically saved and can be resumed later
- Responsive layout suitable for desktop and mobile browsers
- Simple and intuitive user interface with smooth tile movements

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Select a difficulty level and start playing.
